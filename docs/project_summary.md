# Spotify Analytics Platform Project

This project demonstrates the design and implementation of a modern cloud-based analytics platform using Snowflake.

The goal is to simulate how a music streaming company like Spotify could ingest, process, transform, validate, and analyse large-scale listening behaviour data.

## Core Areas Covered

- Data warehousing
- SQL transformations
- ETL/ELT concepts
- Data quality validation
- Analytics-ready modelling
- Cloud architecture thinking
- Business-focused data engineering

## Technologies

- Snowflake
- SQL
- Python
- GitHub
