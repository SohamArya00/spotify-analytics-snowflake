# Spotify-Style Analytics Platform using Snowflake

## Overview

This project demonstrates how I would design and build a modern cloud-based analytics platform for a Spotify-style music streaming product using Snowflake.

The platform simulates ingestion, transformation, validation, and analytics processing for large-scale listening behaviour data.

The goal of this project is to showcase cloud data engineering concepts including:

- ETL/ELT pipelines
- Data warehousing
- SQL transformations
- Analytics modelling
- Data quality validation
- Cloud architecture design
- Business-focused data engineering
